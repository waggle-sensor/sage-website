### Overview

The Summer Camp will provide hands-on learning and experience building novel AI pipelines for Sage AI-enabled instrumentation, measuring and exploring ecosystems and natural hazards with Sage, using Sage data, and extending Sage with new sensors and capabilities.

#### Prerequisite Skills

- Proficiency with SSH and basic Linux shell and command line tools
- Familiarity with using AI models on your laptop via tools such as Ollama or LM Studio
- Familiarity with developing Python applications

#### Required Supplies

- A laptop configured for local use of Python toolchains and AI models

### Deep Dive Topics

#### Sage System Software
Overview of the complete Sage software stack, from booting the node to running Kubernetes containers. Participants will build containers for the edge, build applications that use inference as a service, and connect automated cloud workflows for data.

#### AI-enabled Code Generation & Sage LLM Tools
Learn how to use AI to develop and operate Sage nodes, from writing code to managing nodes with tools such as Cursor, MCP, Claude Code, and more.

#### AI Enablement
The value of Sage depends on reliable and accurate AI models for the edge, including Bioclip2, YOLO, flood detection, cloud motion vectors, wildfire detection, and more.  We will walk through each of the currently available AI tools, and explore how to use and test them, discuss best practices, explore new models from HuggingFace, and deploy new prototypes for domain science teams to try.

#### Data and NSF Resources
Explore and use Sage data tools, from data collected by Beehive to integration and fusion with other sources such as satellite data. Topics include: DOIs, FAIR data catalogs, Pelican, NEON data, workflow engines, and integration of Sage with NSF resources (TACC, NDP, NRP, etc).

#### AI Domain Science
A deep dive into projects using Sage and what they need next. Projects include the Colorado Drought Network, Hawaii Flood Network, Atmospheric Science, and more. The goal is to add complete new integrations and AI capabilities.

#### Hardware and Sensors
Bring your sensors and new hardware. We will work together to add new sensors including infrasound, seismic, hyperspectral cameras, drones, robotic arms, and HaLow cameras. A complete walk-through of what is possible, from LoRaWAN bits to actuators will be provided. Sessions will be hands-on with the expectation that new sensors will be added, documented, tested, and deployed.

#### AI Agents at the Edge
Learn how to build and deploy AI agents for the edge. AI agents are capable, exciting, and potentially dangerous—steering PTZ cameras, giving drones instructions, adjusting operating parameters in nodes. We expect this session to end with dozens of agents working on Sage.

#### FutureTech
Next-generation hardware, tools, and directions

## Hackathon Projects

Attendees will be encouraged to organize into teams and tackle projects with guidance and support from the Sage team.

### Image Search at the Edge
Key pieces: translate the NRP Kubernetes config into a pluginctl setup, swap in edge-friendly models (e.g., Gemma4), replace the vector DB with something edge-compatible (e.g., NVIDIA's NanoDB), and run benchmarks. Creative ideas: pivot from "image search" to generating daily or weekly summaries of what the node sees.

### Projects to Complete During the Week

- **Get new code into the ECR and run it on a mountaintop**
- **Process a large set of images with an LLM and report results to Sage Beehive**
- **Build a pipeline with three time-series linked sensors and analyze with AI in real-time**
